# Carpeta Results

Esta carpeta almacena los resultados de los entrenamientos.

## Archivos generados:

- `training_final_results.png` - Gráficas finales del entrenamiento
- `generation_X_details.png` - Análisis detallado de cada generación
- `training_comparison.png` - Comparación de múltiples entrenamientos
- `game_statistics.png` - Estadísticas de juegos

## Organización:

Puedes crear subcarpetas por fecha o experimento:
```
results/
├── 2025-10-21_experiment_1/
│   ├── training_final_results.png
│   └── generation_*.png
└── 2025-10-22_experiment_2/
    └── ...
```
